Movie Database SQL Project

Project Overview:
This project is a simple SQL database script created as part of my coursework at Extended School. It demonstrates how to populate a database table with movie data and perform basic data retrieval and sorting.
Features
Data Insertion: Adds 11 different movies across various genres (Horror, Romance, Crime, Adventure, Comedy, and Sci-Fi).
Data Organization: Captures key details for each movie, including:
Title
Genre
Release Year
IMDb-style Rating
Sorting Logic: Includes a query to display the entire movie list in alphabetical order by title.
Included Movies
The database includes entries for films such as:
Inception
The Godfather
Moana 2
Titanic
The Shawshank Redemption
How to Use
This .sql file can be imported into any SQL-compatible database manager (like MySQL, PostgreSQL, or SQLite) to recreate the movie table and view the sorted results.
